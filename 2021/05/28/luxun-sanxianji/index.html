读鲁迅《三闲集》《二心集》

发表于 2021-05-28 | 分类于 文学 |
字数统计: 873 | 阅读时长 ≈ 3

1. 文艺与革命


最先进的思想，只有站在最高层的少数人能够了解。芸芸众生，到底有一部分感觉不敏的。


世界文化的推进，全赖少数先觉之冲锋陷阵，如果各个人的聪明才智，都是相等，文化也早就发达到极致了，世界也就大同了。


艺术是文化的一部，文化有进退，艺术自不能除外。民众化的艺术，以艺术本身有无限的价值等差来说，简直不能成立。


2. 现今的新文学的概观


各种文学，都是应环境而产生的，推崇文艺的人，虽喜欢说文艺足以煽起风波来，但在事实上，却是政治先行，文艺后变。


3. 鲁迅译著书目


对于只想以笔墨问世的青年，我现在却敢据几年的经验，以诚恳的心，进一个苦口的忠告。那就是：不断的努力一些，切勿想以一年半载，几篇文字和几本期刊，便立了空前绝后的大勋业。还有一点是：不要只用力于抹杀别个，使他和自己一样空无，而必须跨过那站着的前人，比前人更高大。初初出阵的时候，幼稚和浅薄都不要紧，然而，也须不断的生长起来才好。并不明白文艺的理论而任意作些造谣生事的评论，写几句闲话便要扑灭异己的短评，译几篇童话就想抹杀一切的翻译，归根结蒂，于己于人，还都是“可怜无益费精神”的事，这也就是所谓的“聪明误”了。


世界绝不和我同死，希望是在于将来的。

4. 习惯与改革


体制和精神都已硬化了的人民，对于极小的一点改革，也无不加以阻挠，表面上好像恐怕于自己不便，其实是恐怕于自己不利，但所设的口实，却往往见得极其公正而且堂皇。


现在已不是在书斋中，捧书本高谈宗教、法律、文艺、美术等等的时候了，即使要谈论这些，也必须先知道习惯和风俗，而且有正视这些黑暗面的勇猛和毅力。因为倘不看清，就无从改革，仅大叫未来的光明，其实是欺骗怠慢的自己和怠慢的听众的。

5. 对于左翼作家联盟的意见


倘不明白革命的实际情形，也容易变成“右翼”。革命是痛苦的，其中也必然混有污秽和血，绝不是如诗人所想的那般有趣，那般完美，那般浪漫。革命当然有破坏，然而更需要建设，破坏是痛快的，但建设却是麻烦的事。所以对于革命抱着浪漫蒂克的幻想的人，一旦和革命接近，到革命进行，便容易失望。

6. 阿Q正传（《呐喊》）


有人说：有些胜利者，愿意敌手如虎，如鹰，他才感到胜利的欢喜；假使如羊，如小鸡，他便反觉得胜利的无聊。又有些胜利者，当克服一切之后，看见死的死了，降的降了，“臣诚惶诚恐死罪死罪”，他于是没有了敌人，没有了对手，没有了朋友，只有自己在上，一个，孤另另，凄凉，寂寞，便反而感到了胜利的悲哀。

鲁迅 杂文
